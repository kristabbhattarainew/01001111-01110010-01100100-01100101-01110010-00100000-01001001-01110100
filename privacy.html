Information we collect

When you use the Order It, we collect the following types of information from you:

  • Once you add Order It, Your serverinfo like server name, server id, owner id, server member size will be send to our official server for the bot security and smoothness.
  • Order It uses both prefix and slash command so for prefix commands we listen for the message and filter it out for the commands.
  • All server modules questions are stored in our database.
