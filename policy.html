Privacy Policy for Order It

Last Updated: 14 April 2023

At Order It, accessible from discord, one of our main priorities is the privacy of our users. This Privacy Policy document contains types of information that is collected and recorded by Order It and how we use it.

If you have additional questions or require more information about our Privacy Policy, do not hesitate to contact us in Nepal Studio [Discord Server].

This Privacy Policy applies only to our online activities and is valid for users to our Order It with regards to the information that they shared and/or collect in Order It.

Consent

By using Order It, you agree to this Privacy Policy and give us your consent to collect and use your information in accordance with this policy.

Information we collect

When you use the Order It, we collect the following types of information from you:

  • Once you add Order It, Your serverinfo like server name, server id, owner id, server member size will be send to our official server for the bot security and smoothness.
  • Order It only uses slash command(/) so no message content is used.
  • All server modules questions are stored in our database.
  • Premium user and server data are recorded in our database for filtering the user and server in some bot features
  • Server shop owner id & shop info are stored in our database.
  • No message content is recorded in our database except questions, shop info & server note.

How we use your information

We use the information we collect in various ways, including to:

  • Provide, operate, and maintain Order It features in respective servers
  • Improve, personalize, and expand Order It customization
  • Make Order It user friendly and easy to use
  • Secure Order It and its users privacy

Log Informations

Order It follows a standard procedure of logging some informations. Some of the informations we log are users voting for Order It in top.gg and server info when Order It joins a new server

Sharing Of Your Informations

All your data is stored in our bot database, and we don't provide your information to anyone.

Security

We have hosted Order It on a secure platform with full security measures in place. We have done our best to ensure the best security for both the bot and its data. The owner account is 2FA proof.

Request access to the personal information that we have about you.

Request that we delete any data about you that we have collected.

If you make a request, we will respond within one month. To exercise any of these rights, please contact us on our Official Discord Server at Nepal Studio.
